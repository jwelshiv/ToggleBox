ToggleBox

ToggleBox is a jQuery plugin that turns checkboxes into the stylized toggle switches. ToggleBox simply hides the checkboxes it replaces, and changes their state based on the state of the switch, so forms still function as expected.

Usage

ToggleBox is as easy as any jQuery plugin. Here is the code that runs the example above.

$(document).ready(function(){
	$('input[name="standard"]').toggleBox();
	$('input[name="yesno"]').toggleBox({onLabel:'Yes', offLabel:'No'});
});
